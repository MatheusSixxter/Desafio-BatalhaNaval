# 🚢 Desafio Batalha Naval

Este projeto é uma versão simples do clássico jogo Batalha Naval, feito em Python para rodar no terminal. Desenvolvido para a matéria Introdução à Programação de Computadores.

## 🎯 Objetivo

- Encontre o navio escondido em um tabuleiro 5x5.
- Você tem 5 tentativas para acertar.
- Cada jogada revela se você acertou ou errou (água).
